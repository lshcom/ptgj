01. 파이썬이란?
파이썬(Python)은 세계에서 가장 인기 있는 프로그래밍 언어 중 하나입니다. 읽기 쉽고, 배우기 쉽고, 강력합니다.

파이썬이 뭔가요?

프로그래밍 언어란 컴퓨터에게 명령을 내리는 언어입니다. 우리가 한국어나 영어로 대화하듯, 컴퓨터에게는 파이썬 같은 언어로 말을 걸어요.

🐍 파이썬은 1991년 귀도 반 로섬이 만든 언어로, 문법이 영어와 비슷해서 초보자도 쉽게 배울 수 있습니다.

파이썬으로 무엇을 할 수 있나요?

02. 변수와 데이터 타입
변수는 데이터를 저장하는 상자입니다. 파이썬에는 다양한 종류의 데이터가 있어요.

변수란?

변수(Variable)는 데이터를 담아두는 이름표가 달린 상자입니다. = 기호로 값을 저장해요.

Python
# 변수에 값 저장하기 name = "김파이썬" # 문자열 age = 20 # 정수 height = 175.5 # 소수 is_student = True # 참/거짓 print(name) print(age) print(height) print(is_student)

데이터 타입 4가지

타입이름예시설명
str문자열"안녕", 'hello'텍스트 데이터
int정수10, -5, 0소수점 없는 숫자
float실수3.14, -0.5소수점 있는 숫자
bool불리언True, False참 또는 거짓
Python
# type() 로 타입 확인하기 print(type("안녕")) # <class 'str'> print(type(42)) # <class 'int'> print(type(3.14)) # <class 'float'> print(type(True)) # <class 'bool'>

✅영문자, 숫자, 밑줄(_) 사용 가능 · 숫자로 시작 불가 · 대소문자 구분
⚠️1name, my-var, class(예약어) 등은 사용 불가!

03. 출력과 입력
사용자와 소통하는 방법을 배웁니다. print()로 출력하고, input()으로 사용자의 입력을 받을 수 있어요.

print() 완전 정복

Python
# 여러 값 출력 (쉼표로 구분) name = "이수진" age = 25 print("이름:", name, "나이:", age) # f-string (가장 많이 쓰는 방법!) print(f"안녕하세요, {name}님! 나이는 {age}살이네요.") # end, sep 옵션 print("사과", "바나나", "포도", sep=" | ") print("끝에 느낌표", end="!\n")
💡 f-string: 문자열 앞에 f를 붙이고 중괄호 {} 안에 변수를 넣으면 값이 출력됩니다!

input() - 사용자 입력 받기

Python

04. 연산자
파이썬에서 사용할 수 있는 다양한 연산자를 배웁니다. 수학 계산부터 비교, 논리 연산까지!

산술 연산자

Python
a, b = 10, 3 print(a + b) # 13 (더하기) print(a - b) # 7 (빼기) print(a * b) # 30 (곱하기) print(a / b) # 3.3333... (나누기) print(a // b) # 3 (몫) print(a % b) # 1 (나머지) print(a ** b) # 1000 (거듭제곱)

비교 연산자

두 값을 비교해서 True 또는 False를 반환합니다.

Python
print(5 == 5) # True (같다) print(5 != 3) # True (다르다) print(5 > 3) # True (크다) print(5 < 3) # False (작다) print(5 >= 5) # True (크거나 같다) print(5 <= 4) # False (작거나 같다)

논리 연산자

Python
print(True and False) # False (둘 다 True여야 True) print(True or False) # True (하나만 True여도 True) print(not True) # False (반대로!) # 실제 사용 예시 age = 20 print(age >= 18 and age <= 65) # True (성인)

A0
B1

D3.33
✅ 정답! %는 나머지 연산자입니다. 10을 3으로 나누면 몫은 3, 나머지는 1이에요.
05. 조건문 (if/else)
조건에 따라 다른 코드를 실행하는 방법입니다. 프로그래밍의 핵심 개념!

if / elif / else 구조

Python
age = 17 if age >= 18: print("성인입니다.") elif age >= 14: print("청소년입니다.") else: print("어린이입니다.")
⚠️ 파이썬에서는 들여쓰기(indent)가 매우 중요합니다! if 아래 실행할 코드는 반드시 4칸 들여쓰기를 해야 해요.

중첩 조건문

Python
score = 85 if score >= 90: print("A학점") elif score >= 80: print("B학점") elif score >= 70: print("C학점") else: print("F학점") # 출력: B학점
✏️ PRACTICE
온도를 변수로 저장하고, 30도 이상이면 "더워요!", 10도 이하면 "추워요!", 그 사이면 "적당해요!" 를 출력하는 코드를 작성하세요.

06. 반복문 (for/while)
같은 작업을 여러 번 반복할 때 사용하는 반복문! 코딩을 강력하게 만드는 핵심입니다.

for 반복문

Python
# range()로 숫자 반복 for i in range(5): # 0, 1, 2, 3, 4 print(f"{i}번째 반복!") # 리스트 반복 fruits = ["사과", "바나나", "포도"] for fruit in fruits: print(f"나는 {fruit}을 좋아해요!") # range(시작, 끝, 간격) for i in range(1, 10, 2): # 1, 3, 5, 7, 9 print(i, end=" ")

while 반복문

Python
# 조건이 True인 동안 반복 count = 1 while count <= 5: print(f"카운트: {count}") count += 1 # count = count + 1 # break와 continue for i in range(10): if i == 5: break # 반복 완전 종료 if i % 2 == 0: continue # 이번 반복 건너뛰기 print(i)

07. 리스트와 튜플
여러 개의 데이터를 하나로 묶어 관리하는 자료구조! 파이썬에서 가장 많이 쓰는 개념입니다.

리스트 (List)

리스트는 여러 값을 순서대로 저장하는 자료구조입니다. 대괄호 []로 만들어요.

Python
fruits = ["사과", "바나나", "포도", "딸기"] # 인덱싱 (0부터 시작!) print(fruits[0]) # 사과 print(fruits[-1]) # 딸기 (마지막) # 슬라이싱 print(fruits[1:3]) # ['바나나', '포도'] # 리스트 수정 fruits.append("망고") # 끝에 추가 fruits.insert(1, "키위") # 1번 위치에 삽입 fruits.remove("바나나") # 해당 값 삭제 fruits.pop() # 마지막 값 제거 print(len(fruits)) # 길이 print(sorted(fruits)) # 정렬

튜플 (Tuple)

튜플은 리스트와 비슷하지만 변경이 불가능합니다. 괄호 ()로 만들어요.

Python
point = (10, 20) # x, y 좌표 print(point[0]) # 10 # 튜플 언패킹 x, y = point print(f"x={x}, y={y}") # point[0] = 30 # 오류! 튜플은 변경 불가
💡리스트는 데이터를 자주 바꿀 때, 튜플은 바뀌지 않아야 할 데이터(좌표, 날짜 등)에 사용합니다.

08. 딕셔너리와 집합
키-값 쌍으로 데이터를 저장하는 딕셔너리, 중복 없는 집합! 실무에서 자주 쓰이는 자료구조입니다.

딕셔너리 (Dictionary)

Python
person = { "name": "김파이썬", "age": 25, "city": "서울" } # 값 읽기 print(person["name"]) # 김파이썬 print(person.get("age")) # 25 (없으면 None) # 값 추가/수정 person["job"] = "개발자" person["age"] = 26 # 반복 for key, value in person.items(): print(f"{key}: {value}") # 키 목록, 값 목록 print(person.keys()) print(person.values())

집합 (Set)

Python
nums = {1, 2, 3, 2, 1} print(nums) # {1, 2, 3} 중복 제거! # 집합 연산 a = {1, 2, 3, 4} b = {3, 4, 5, 6} print(a | b) # 합집합: {1,2,3,4,5,6} print(a & b) # 교집합: {3,4} print(a - b) # 차집합: {1,2}

12. 클래스와 객체 (OOP)
객체지향 프로그래밍(OOP)! 현실 세계를 코드로 모델링하는 강력한 패러다임입니다.

클래스란?

클래스는 객체를 만드는 설계도입니다. 예를 들어 "강아지"라는 클래스를 만들면 여러 강아지 객체를 만들 수 있어요.

Python
class Dog: # 생성자: 객체가 만들어질 때 실행 def __init__(self, name, breed): self.name = name # 속성(attribute) self.breed = breed # 메서드(method) def bark(self): print(f"{self.name}: 멍멍!") def info(self): print(f"이름: {self.name}, 품종: {self.breed}") # 객체 생성 dog1 = Dog("멍멍이", "포메라니안") dog2 = Dog("초코", "푸들") dog1.bark() dog2.info()

Python

프로젝트 1: 간단한 계산기

Python
def calculator(): print("=== 간단한 계산기 ===") try: a = float(input("첫 번째 숫자: ")) op = input("연산자 (+,-,*,/): ") b = float(input("두 번째 숫자: ")) if op == "+": result = a + b elif op == "-": result = a - b elif op == "*": result = a * b elif op == "/": if b == 0: raise ZeroDivisionError result = a / b else: print("올바른 연산자를 입력하세요.") return print(f"결과: {a} {op} {b} = {result}") except ZeroDivisionError: print("0으로 나눌 수 없습니다!") except ValueError: print("숫자를 올바르게 입력하세요!") calculator()

프로젝트 2: 성적 관리 프로그램

Python
class GradeManager: def __init__(self): self.students = {} def add_score(self, name, score): self.students[name] = score def get_grade(self, score): if score >= 90: return "A" elif score >= 80: return "B" elif score >= 70: return "C" else: return "F" def report(self): print("\n📊 성적표") print("-" * 30) for name, score in self.students.items(): grade = self.get_grade(score) print(f"{name:6} | {score:3}점 | {grade}학점") avg = sum(self.students.values()) / len(self.students) print(f"\n평균: {avg:.1f}점") # 실행 gm = GradeManager() gm.add_score("김철수", 92) gm.add_score("이수진", 78) gm.add_score("박민준", 85) gm.report()
